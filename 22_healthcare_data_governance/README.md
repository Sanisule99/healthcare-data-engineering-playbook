# 22 — Healthcare Data Governance

This module introduces healthcare data governance concepts for informatics, analytics, data engineering, and AI workflows.

## Purpose

Strong healthcare data systems require more than pipelines and models. They require governance.

This includes:

- data ownership
- stewardship
- data dictionaries
- PHI protection
- access controls
- lineage tracking
- reproducibility
- responsible AI documentation
