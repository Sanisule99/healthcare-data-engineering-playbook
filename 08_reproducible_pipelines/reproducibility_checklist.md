# Reproducibility Checklist

Before sharing or rerunning a healthcare analytics pipeline:

- [ ] Input files are documented
- [ ] Required columns are validated
- [ ] Configuration is stored separately from code
- [ ] Outputs are saved consistently
- [ ] Logs are generated
- [ ] Data privacy is reviewed
- [ ] No PHI is uploaded
- [ ] Pipeline can be rerun from a clean environment
- [ ] Dependencies are documented
