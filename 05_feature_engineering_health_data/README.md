# 05 — Feature Engineering for Health Data

This module demonstrates how to create useful analytic features from raw healthcare data.

## Why This Matters

Healthcare models and dashboards often require derived features such as:

- age groups
- high utilization flags
- chronic condition burden
- length-of-stay categories
- risk scores
- screening completion rates
- follow-up indicators

Feature engineering helps transform raw clinical records into analysis-ready variables.

## Files

| File | Purpose |
|---|---|
| feature_engineering_demo.py | Creates derived healthcare features |
| data/raw_health_features.csv | Synthetic raw health data |
| outputs/engineered_features.csv | Analysis-ready output |
| feature_dictionary.md | Definitions of engineered features |

## Skills Demonstrated

- Data transformation
- Clinical feature creation
- Rule-based risk indicators
- Analytics-ready datasets
- Reproducible preprocessing
