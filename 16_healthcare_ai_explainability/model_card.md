# Model Card — Synthetic Healthcare Risk Model

## Model Type

Random Forest Classifier

## Intended Use

Educational demonstration of healthcare AI explainability.

## Not Intended For

Clinical decision-making.

## Training Data

Synthetic patient-level risk dataset.

## Target

Synthetic high-risk outcome.

## Features

- age
- prior encounters
- chronic condition count
- abnormal lab flag
- medication count

## Limitations

- Synthetic data
- Small sample size
- No external validation
- No fairness audit yet
- No clinical deployment testing

## Responsible Use

This model should only be used for learning and demonstration.
