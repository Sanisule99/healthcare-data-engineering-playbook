# 01 — Healthcare Data Sources

This module introduces common healthcare data sources used in analytics, informatics, data engineering, public health, and AI.

## Common Data Sources

- Electronic Health Records
- Claims data
- Laboratory data
- Pharmacy data
- FHIR APIs
- Public health datasets
- Imaging data
- Genomics data
- Patient-reported outcomes
- Synthetic datasets
