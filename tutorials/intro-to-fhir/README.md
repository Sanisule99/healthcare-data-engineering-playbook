# Intro to FHIR APIs with Python

This beginner-friendly tutorial introduces FHIR APIs and demonstrates how to retrieve and parse healthcare data using Python.

## What You Will Learn

- What FHIR is and why it matters in healthcare data engineering
- How to query a public FHIR server
- How to retrieve Patient and Observation resources
- How to convert FHIR JSON into analysis-ready tables
- How FHIR supports interoperability and AI-ready healthcare workflows

## Files

| File | Description |
|---|---|
| `intro_to_fhir_python.py` | Python script for querying a public FHIR API |
| `fhir_patient_observation_walkthrough.md` | Step-by-step explanation |

## Requirements

```bash
pip install requests pandas
```

## Data Source

This tutorial uses a public sandbox FHIR server for educational purposes. Do not use real patient data in this tutorial.
