# 04 — Missing Clinical Data

This module demonstrates how to identify, summarize, and handle missing values in healthcare datasets.

## Why This Matters

Clinical data are often incomplete because of:

- optional documentation fields
- fragmented care across systems
- incomplete screening workflows
- extraction errors
- inconsistent coding
- unavailable lab or follow-up data

Understanding missingness is essential before building dashboards, models, reports, or quality measures.

## Files

| File | Purpose |

| missing_data_analysis.py | Runs missingness analysis and simple imputation |
| data/clinical_missing_data_sample.csv | Synthetic clinical dataset with missing values |
