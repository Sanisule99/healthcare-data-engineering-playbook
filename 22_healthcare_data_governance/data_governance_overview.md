# Healthcare Data Governance Overview

## Why governance matters

Healthcare data are sensitive, complex, and often used for decisions that affect patients, programs, and systems.

Governance ensures data are:

- understood
- protected
- documented
- reproducible
- fit for purpose
- used responsibly

## Core Governance Domains

### 1. Data Ownership

Who is responsible for the dataset?

### 2. Data Stewardship

Who maintains data quality and documentation?

### 3. Data Dictionary

What does each variable mean?

### 4. Data Lineage

Where did the data come from and how was it transformed?

### 5. Privacy and PHI Review

Does the dataset contain identifiers or sensitive fields?

### 6. Access Control

Who should be able to view, modify, or export data?

### 7. Model Governance

If data are used for AI/ML, how is model performance, bias, and drift monitored?

## Practical Rule

If the dataset cannot be explained, traced, or safely shared, it is not ready for analytics or modeling.
