# 20 — Public Health Informatics

This module demonstrates how informatics workflows can support public health program monitoring, quality improvement, and population health analytics.

## Purpose

Public health informatics helps transform program and community-level data into actionable insights.

This module uses synthetic data to demonstrate:

- measure calculation
- numerator / denominator logic
- site-level performance summaries
- equity stratification
- public health dashboard preparation

## Files

| File | Purpose |
|---|---|
| public_health_metrics_analysis.py | Runs metric calculations |
| data/public_health_program_data.csv | Synthetic public health dataset |
| measure_dictionary.md | Measure definitions |
| outputs/ | Generated summary outputs |

## Skills Demonstrated

- Public health metric design
- Numerator / denominator thinking
- Program evaluation analytics
- Equity stratification
- Reproducible reporting
