# 06 — Imbalanced Medical Datasets

This module demonstrates how to evaluate and model imbalanced healthcare datasets.

## Why This Matters

Many important healthcare outcomes are rare, including:

- mortality
- readmission
- sepsis
- adverse events
- disease recurrence
- severe maternal morbidity
- rare diagnoses

If handled incorrectly, models may appear accurate while failing to detect the minority outcome.

## Topics Covered

- Class imbalance
- Accuracy vs recall
- Confusion matrix
- Precision and recall
- ROC AUC
- Class weighting
- Responsible interpretation
