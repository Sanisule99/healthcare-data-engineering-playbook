# PHI Review Checklist

Use this checklist before sharing, uploading, or analyzing healthcare data.

## Direct Identifiers

- [ ] Names removed
- [ ] Street addresses removed
- [ ] Phone numbers removed
- [ ] Email addresses removed
- [ ] Social Security numbers removed
- [ ] Medical record numbers removed
- [ ] Account numbers removed
- [ ] Certificate/license numbers removed
- [ ] Device identifiers removed
- [ ] URLs/IP addresses removed
- [ ] Full-face images removed

## Dates and Locations

- [ ] Exact dates reviewed
- [ ] Small geographic units reviewed
- [ ] Rare event combinations reviewed

## Dataset Review

- [ ] Data source documented
- [ ] Data owner identified
- [ ] Intended use documented
- [ ] Sharing permissions confirmed
- [ ] Public release reviewed

## Final Decision

- [ ] Safe for public sharing
- [ ] Internal use only
- [ ] Requires further privacy review
