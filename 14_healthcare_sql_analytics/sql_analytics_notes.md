# SQL Analytics Notes

Healthcare SQL workflows commonly include:

- Cohort definition
- Utilization analysis
- Length of stay metrics
- Readmission analysis
- Diagnosis frequency analysis
- Claims aggregation
- Patient-level joins

- diagnoses
- procedures
- medications
- claims

- CASE WHEN
- Window functions
- CTEs
- Aggregations
